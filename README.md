# 🌍 Personalized Tour Recommendation System With Real-Time Chat

A smart travel recommendation platform that suggests personalized tour destinations based on user interests and provides real-time chat interaction for better user engagement.
---
## 🚀 Features

- 🔍 Personalized tour recommendations
- 💬 Real-time chatbot interaction
- 🎨 Modern responsive UI
- ⚡ Fast frontend using Vite + React
- 📱 Mobile-friendly design
- 📊 Recommendation dashboard
- 🔐 Backend API integration support
- 🌐 REST API architecture

---

## 🛠️ Tech Stack

### Frontend
- React
- TypeScript
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Other Tools
- JWT Authentication
- REST APIs
- Nodemon

## ⚙️ Installation & Setup
### 1️⃣ Clone Repository
```bash
git clone https://github.com/your-username/A-Personalized-Tour-Recommendation-System-With-Real-Time-Chat.git
```
---
### 2️⃣ Open Project
```bash
cd A-Personalized-Tour-Recommendation-System-With-Real-Time-Chat
```
---
## ▶️ Frontend Setup

Install frontend dependencies:

```bash
npm install
```

Run frontend:

```bash
npm run dev
```

Frontend runs on:

```bash
http://localhost:5173
```
---
## ▶️ Backend Setup

Move to backend folder:

```bash
cd backend
```

Install dependencies:

```bash
npm install
```
Run backend server:
```bash
npm run dev
```
Backend runs on:
```bash
http://localhost:5000
```
---
## 🗄️ MongoDB Setup

Install MongoDB Community Server:
https://www.mongodb.com/try/download/community
Create `.env` file inside backend folder:
```env
PORT=5000
MONGODB_URI=mongodb://127.0.0.1:27017/tourrecommendation
JWT_SECRET=your_secret_key
```
